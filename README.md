## 📚 研究主题

本项目聚焦于**氧化应激（Oxidative Stress）**在**纤维化（Fibrosis）**发生发展中的作用机制，包括：

- **直接机制**: ROS直接诱导成纤维细胞活化、EMT、ECM沉积
- **间接机制**: 通过免疫细胞（巨噬细胞、中性粒细胞等）介导的纤维化
- **分子通路**: NLRP3炎症小体、NADPH氧化酶、线粒体功能障碍等
- **治疗策略**: 抗氧化治疗、靶向干预
