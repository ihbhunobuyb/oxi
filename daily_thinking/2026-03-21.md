## 核心见解

### 1. 巨噬细胞是氧化应激-纤维化的关键桥梁
SLC15A3研究揭示：巨噬细胞中的氧化应激状态直接决定纤维化进程，为靶向治疗提供新靶点。

### 2. Nrf2是核心抗氧化调控因子
多个研究表明：Nrf2通路激活可以抑制氧化应激诱导的纤维化，是重要的治疗靶点。

### 3. 机械敏感性通道与代谢的交叉
Piezo1的最新研究揭示：机械信号可以通过氧化应激通路影响纤维化。

### 4. 线粒体功能障碍是核心环节
Akt1介导的线粒体自噬研究表明：维持线粒体功能可能是阻止纤维化的关键。

## 关键机制总结

```
氧化应激 → NLRP3炎症小体 → IL-1β/IL-18 → 成纤维细胞活化 → 纤维化
     ↓
  线粒体功能障碍 → ROS累积 → 巨噬细胞极化(M1/M2) → 促纤维化微环境
     ↓
  Nrf2通路抑制 → 抗氧化能力下降 → 氧化损伤累积 → EMT/成纤维细胞活化
```
